Ví dụ về HTTP
qBạn đánh địa chỉ trên trình duyệt
qwww.someSchool.edu/someDepartment/home.index
q1a. HTTP client khởi tạo kết nối TCP tới HTTP server tại địa chỉ www.someSchool.edu. Cổng mặc định là 80.
q2. HTTP client gửi thông điệp HTTP yêu cầu (bao gồm URL) qua kết nối TCP vừa thiết lập
q1b. HTTP server ở địa chỉ www.someSchool.edu đợi kết nối TCP ở cổng 80,  chấp nhận kết nối, thông báo lại cho client.
q3. HTTP server nhận thông điệp yêu cầu,lấy các đối tượng được yêu cầu gửi vào trong thông điệp trả lời, (someDepartment/home.index) gửi thông điệp vào socket
