rDHCP: Dynamic Host Configuration Protocol: tự động xin cấp phát địa chỉ theo kiểu “plug-and-play”
mMáy tính quảng bá thông điệp “DHCP discover”
mDHCP server trả lời với thông điệp “DHCP offer”
mMáy tính yêu cầu địa chỉ IP bằng thông điệp “DHCP request”
mDHCP server gửi địa chỉ qua thông điệp “DHCP ack”
