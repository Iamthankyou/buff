ICMP: Internet Control Message Protocol
rĐược các Máy tính, Router, Gateway sử dụng để trao đổi các thông tin về tầng Mạng
mBáo lỗi: unreachable host, network, port, protocol
mHiển thị request/reply (Lệnh ping)
rTrong hệ thống “nằm trên” IP:
mThông điệp ICMP được đặt trong IP datagram
rThông điệp ICMP : Kiểu (Type), Mã (code) cùng với 8 byte đầu tiên của IP datagram gây lỗi
Type  Code  description
0        0         echo reply (ping)
3        0         dest. network unreachable
3        1         dest host unreachable
3        2         dest protocol unreachable
3        3         dest port unreachable
3        6         dest network unknown
3        7         dest host unknown
4        0         source quench (congestion
                     control - not used)
8        0         echo request (ping)
9        0         route advertisement
10      0         router discovery
11      0         TTL expired
12      0         bad IP header
