Lập trình TCP Socket
qClient phải liên lạc với server
qTiến trình trên server phải chạy trước. qServer phải tạo sẵn socket (door) để tiếp nhận yêu cầu từ client.
qClient trao đổi với server bằng cách:
qTạo socket TCP ở phía client
qXác định địa chỉ IP, số hiệu cổng của tiến trình server.
qKhi client tạo socket: client TCP thiết lập kết nối tới server TCP.
qKhi nhận được yêu cầu từ client, server TCP tạo socket mới cho tiến trình trên server trao đổi dữ liệu với client
oCho phép server có thể đáp ứng yêu cầu của nhiều client.
Quan điểm Lập trình Ứng dụng
TCP cung cấp dịch vụ truyền dữ liệu tin cậy theo byte giữa Client và Server
