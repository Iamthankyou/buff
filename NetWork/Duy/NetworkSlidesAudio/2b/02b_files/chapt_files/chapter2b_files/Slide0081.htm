Bản ghi DNS
qDNS: cơ sở dữ liệu phân tán lưu các Bản ghi Tài nguyên (RR)
qType=NS
oname : domain (ví dụ foo.com)
ovalue : địa chỉ IP của authoritative name server ứng với miền đó
Định dạng RR : (name, value, type, TTL)
qType=A
oname : hostname
ovalue : IP address
q
qType=CNAME
oname : tên bí danh cho một tên thực nào đó :  ví dụ www.ibm.com là tên bí danh của servereast.backup2.ibm.com
ovalue : tên thực
q
qType=MX
ovalue : tên của mailserver
