DNS : Name Server (Máy chủ Tên)
qKhông có server nào có thể lưu toàn bộ được tên miền và địa chỉ IP tương ứng
qlocal name servers:
oMỗi  ISP, công ty có local name server (ngầm định)
oCâu hỏi truy vấn của host về DNS sẽ được chuyển tới local name server
qChức năng của name server:
oĐối với host: lưu địa chỉ IP và tên miền tương ứng của host
oCó thể tìm tên miền ứng với địa chỉ IP và ngược lại
qTại sao tạo ra một DNS Server tập trung ?
qĐiểm hỏng duy nhất - nếu name-server “chết” thì cả mạng Internet sẽ “chết” theo.
qKhối lượng giao dịch tại điểm tập trung lớn.
qCơ sở dữ liệu tập trung ở “xa” với nhiều nơi
